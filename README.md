A Streamlit-based application that integrates three operational domains—Cybersecurity, Data Science, and IT Operations—into a unified, role-based dashboard. The platform provides secure authentication, CRUD functionality, interactive visualisations, and an AI assistant for contextual support.

---

## Features
### 1. Secure Authentication
- User login and registration
- Password hashing
- Role-based access control (cyber, data, IT)

### 2. Multi-Page Interface
- Cybersecurity dashboard
- Data Science dashboard
- IT Operations dashboard
- AI Assistant page
- Role-specific navigation via sidebar

### 3. CRUD Functionality

- Each domain supports:
- Create new records
- View all records
- Delete existing records
- Real-time data updates

### 4. Plotly Visualisations
- Cybersecurity: Incident severity distribution
- Data Science: Dataset dimensions (rows vs columns)
- IT Operations: Ticket status distribution

### 5. AI Assistant

Provides explanations, insights, and general support using an integrated AI interface

---

## Technologies Used
- Streamlit – UI framework
- Plotly – Interactive charts
- SQLite – Persistent storage
- Python (OOP) – Models and services
- AI Integration – Custom AI assistant module
