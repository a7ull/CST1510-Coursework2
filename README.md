## Project Description
This project implements a **secure authentication system** using Python.  
It allows users to register and log in securely through a command-line interface, with password hashing using the bcrypt library.
---
## Ô∏è Features
- Secure password hashing using bcrypt with automatic salt generation

- File-based user data persistence
---
##  Technical Implementation
- Hashing Algorithm: bcrypt with automatic salting
- Data Storage: Plain text file (`users.txt`) with comma-separated values
- Password Security: One-way hashing, no plaintext storage
- Validation: Username (3-20 alphanumeric characters), Password (6-50 characters).
